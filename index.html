PacmanForUbuntu by sybarite

This a experimental and learning based project to develop pac man like game for ubuntu

This project is a learn and research project created by me, any one can join and pitch in ideas for the project. Though I will be happy if more game programming "noobs" like me will join this project. Send me an email at vivek@epicwhale.org with your git hub account so I can add you as a colloborator.

Install

This project is a ubuntu OS based project, also please install SDL in your ubuntu system to develop this project. Here is a detailed page http://bit.ly/nwbUY9 (don't get afraid of the shortener :p). Please follow that tutorial for a more comprehensive understanding for developing in SDL.
